# Getting Started

This guide walks you through the initial setup of WeftID: your first sign-in as a super admin, adding an application for SSO, and optionally connecting an external identity provider for federated sign-in.

1. [First Login](first-login.md) — Sign in and set your password
2. [Connecting an Identity Provider](connecting-an-idp.md) — Federate with Okta, Entra ID, or another SAML IdP (optional)
3. [Adding an Application](adding-an-application.md) — Register an application for single sign-on
